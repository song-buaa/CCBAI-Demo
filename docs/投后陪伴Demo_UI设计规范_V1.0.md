# 手机银行投后陪伴 Demo — UI设计规范

**版本：V1.0 | 用途：Claude Code开发参考**

---

## 说明

本文档定义 Demo 各页面的 UI 设计要求，以建行手机银行真实截图为视觉基准，在此基础上进行针对性改造。每个场景页面单独一节，逐步补充完整。

**整体视觉风格原则：**

- 以建行手机银行现有 UI 风格为基准，保持银行 App 的专业感和克制感
- **色调层级规则（重要）：**
  - **财富首页**：金色主调（`#C8973A` 近似色），与建行手机银行财富频道保持一致
  - **所有二级页面**（基金持仓页、财富全景页、交易成功页、消息中心、AI对话界面等）：建行蓝主调（`#1677FF` 近似色），与建行严选等二级页面保持一致
- AI 理财助理图标（蓝色圆形 + 白色sparkle）在金色首页和蓝色二级页面中均保持不变
- AI 相关元素用统一样式标识，与普通内容形成区分但不突兀

### 3.3 AI 提示条样式（参考支付宝，颜色用建行蓝）

AI提示条嵌在产品卡片底部，浅蓝色背景（`#EFF4FF` 或近似的极浅蓝），整条可点击。

**内部结构（从左到右）：**

```
[✦sparkle图标]  [类型标签加粗]  [提示文字，超出省略...]
```

| 元素 | 规格 |
|------|------|
| 左侧图标 | 蓝色sparkle（✦），约16px，与文字垂直居中 |
| 类型标签 | **加粗**，建行蓝色（`#1677FF`），取值见下表 |
| 提示文字 | 正常字重，深灰色，跟在类型标签后，超出用「…」截断 |
| 背景色 | 极浅蓝（`#EFF4FF`）或浅灰（`#F5F5F5`），圆角 |
| 整条点击 | 跳转 AI 理财助理对话界面 |

**类型标签取值（对应AI脚本触发类型）：**

| 类型标签 | 对应触发场景 | 示例提示文字 |
|---------|------------|------------|
| 市场解读 | 市场事件、板块波动 | 纳斯达克近期受美联储预期影响大幅波动… |
| 收益提醒 | 盈利里程碑 | 持仓收益已突破+20%，进入历史较高盈利区间… |
| 风险提示 | 亏损超阈值 | 新能源板块近期承压，持仓亏损-15.3%… |
| 投资锦囊 | 常规陪伴、科普 | 红利低波逆势吸金！近60日资金净流入超40亿… |
